A quick demo on JavaScript expressions

Vue only allows expressions to be evaluated and executed. Statements are not supported.

The expression used to determine a person's age based on their date of birth is as follows:

{{ (d && m && y) ? ( Math.floor((Date.now() - new Date(y + "/" + m + "/" + d)) / 31557600000) ? Math.floor((Date.now() - new Date(y + "/" + m + "/" + d)) / 31557600000) + " yrs old." : "Failing Validation" ) : "Failing Validation" }}

VueJS Code Samples

This page displays a list of live code samples for Vue JS (version 2.1.3).

I'm currently training myself on coding with Vue JS, the code samples below will evolve.

This very page is powered by Vue JS and I keep upgrading and updating it as I learn.

I am using the Axios library (version 0.15.2) for the Ajax call. This page will not work with IE version 11 or less. But it does work in MS Edge. Eventually, I will look into why the pages don't work in IE 11, I think it's the Axios library which requires some polyfill for promises.

In some samples, you will see other libraries, anything which gets the job done and complements VueJS is being used.

Note: you need to use a modern desktop browser, I've not attempted to make this mobile friendly.

Note: This is work in progress, there will be new additions and modifications on a regular basis.

Repo: https://github.com/cgauthier/VUEJS_2.1.3
